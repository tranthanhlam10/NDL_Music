## Công nghệ sử dụng 
Ngôn ngữ: Java
* UI: xml
* Môi trường phát triển: Anroid Studio
* Cơ sở dữ liệu: SQLite
* Thiết bị: Điện thoại Android
* SDK version: 25

## Thư viện

* RAM >= 2GB
